About ICP

Internet Computer (ICP)
A blockchain-based protocol that aims to create a decentralized alternative to centralized internet cloud providers:

  • ICP is a network of independent data centers creating a decentralized internet. It uses smart contracts to replace traditional software.
  • Interfaces with other blockchains and Web 2 resources, using languages like Rust and TypeScript.
  • ICP token is used for governance, rewards, and transaction fees.
